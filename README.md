# X15

Repository for the X15 project by [Podlodka AI Club](https://github.com/podlodka-ai-club).

## Overview

This repository is part of the Podlodka AI Club initiative. It currently contains a small framework-free TypeScript storefront skeleton powered by Vite.

## Getting Started

```bash
git clone git@github.com:podlodka-ai-club/X15.git
cd X15
npm install
npm run dev
```

The development server prints a local URL for the storefront preview.

## Verification

Run the local checks before submitting changes:

```bash
npm run type-check
npm run lint
npm run format:check
npm test
npm run build
```
